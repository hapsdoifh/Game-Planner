Organize the excel files as such:
1st column for team names
2nd column for Locations
3rd column for time slots
Starting each column with it's purpose. Ex: 
Team name		Locations		Times
Team1			L1			T1
Team2			L2			T2
...			...			...

Enter the current year into the text box then click on "get file"
Everything will be put into the "Output-Folder"
Make sure you don't have the excel sheets open when you're running the program
